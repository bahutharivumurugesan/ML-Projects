# 🚀 Spam Detection System using Machine Learning

An intelligent Machine Learning based spam classification system that detects whether a message is **Spam 🚨** or **Legitimate ✅** using Natural Language Processing (NLP) and Logistic Regression.

Developed as a beginner-friendly Machine Learning project focused on text classification, prediction analysis, and visualization techniques using Python.

---

# 📌 Project Overview

Spam messages are one of the most common challenges in digital communication systems. This project uses Machine Learning techniques to automatically classify messages as spam or safe.

The system processes textual data, converts it into numerical vectors using **TF-IDF Vectorization**, and predicts message categories using a **Logistic Regression** model.

The project also provides:
- 📊 Prediction Visualization
- 📈 Confidence Analysis
- 🔍 Real-time Message Testing
- 🎯 Interactive Result Dashboard
